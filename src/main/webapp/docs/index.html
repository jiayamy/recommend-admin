关于本手册

媒资管理系统是一个集媒资管理、音视频生产、版权管理、内容管理、直播监控为一体的综合管理系统。包含了从媒资（包括点播和直播）的生产、编辑、审核到发布的整个流程以及生产媒资所需的版权、条目、奖项和人物信息的管理。
