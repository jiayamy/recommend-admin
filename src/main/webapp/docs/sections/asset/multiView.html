直播监控

1.界面截图

直播监控是监控多个直播频道或对某一频道的多个输出源，确保频道播放正常

2.功能说明

2.1 选择监控频道

根据频道名称、频道主分类、电视台类型等多个条件查询频道，可选择一个或多个频道进行监控，默认以最低码率播放多个直播频

2.2 监控单个频道的多个输出源

双击某一频道，可对某一频道的多个输出源进行监控，确保频道播放正常

2.3 移动频道监控位置

在监管管理中，可移动监控频道的位置

2.4 删除监控频道

可删除选中的监管频道

3.操作指南

3.1 搜索申请记录

进入直播监控，点击按钮""可以搜频道列表

3.2 选择监控频道

点击按钮""可以，可对某一频道的多个输出源进行监控，确保频道播放正常

3.3 移动频道监控位置

在监管管理中，可移动监控频道的位置

3.4 删除监控频道

可删除选中的监管频道
