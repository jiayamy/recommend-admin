内容打点

1.界面截图

内容打点是对内容库的视频文件的片头、片尾和精彩部分进行打点。

2.功能说明

2.1 新建打点记录

选择打点的时间点，点击“按钮”，选择打点类型：片头、片尾和精彩看点，填写打点描述

2.2 查看打点记录

点击播放按钮，查看以前的打点记录

2.3 删除打点记录

删除打点记录

2.4 导入打点记录

导入打点记录文件

2.6 导出打点记录

将打点记录导出

2.7 发布打点记录

将打点记录发布到P

3.操作指南

3.1 搜索打点内容

进入内容打点，点击按钮""可以搜索内容列表

3.2 新建打点记录

选择需打点的内容,点击按钮"",选择打点的时间点，点击“按钮”，选择打点类型：片头、片尾和精彩看点，填写打点描述

3.3 查看打点记录

点击按钮""，查看打点记录

3.4 删除打点记录

点击按钮""删除打点记录

3.5 导入打点记录

点击按钮""导入打点记录文件

3.6 导出打点记录

点击按钮""将打点记录导出

3.7 发布打点记录
