奖项任务

1.界面截图

进入我的工作台到我的任务下找到奖项任务，查看奖项媒资申请记录，并可处理奖项任务

2.功能说明

2.1 查看奖项任务

根据奖项名称、奖项状态及提交入库时间等条件查询所有组织的奖项申请记录，双击进入可查看任务详情

2.2 处理奖项任务

双击进入状态为“提交入库”的奖项申请任务详情页面，将该奖项申请直播入库或入库驳回，也可在编辑后将该奖项申请直播入库或入库驳回

2.3 导出奖项申请记录

将搜索出的奖项申请记录导出

3.操作指南

3.1 搜索申请记录

进入奖项申请，点击按钮""可以搜索列表

3.2 搜索奖项申请记录

根据等查询条件搜索此用户所在组织的奖项媒资申请记录，具有高级权限的账户可点击选中“”查看所的组织的申请记录。

3.3 查看奖项申请记录详情

进入奖项申请，双击列表可查看申请详情。

3.4 处理奖项任务

点击按钮""或""可以批量处理媒资任务。或者点击按钮逐个处理任务。

3.5 导出奖项申请记录

点击按钮""将搜索出的奖项申请记录导出
