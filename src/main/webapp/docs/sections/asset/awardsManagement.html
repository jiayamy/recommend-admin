奖项库

1.界面截图

进入媒资管理到资信库再到奖项库，对媒资相关奖项（明星）进行管理，实现对奖项的检索、新建、变更、提交、入库和发布操作。

2.功能说明

2.1 搜索奖项

根据奖项名称、奖项状态及提交入库时间搜索奖项

2.2 查看奖项

双击查看奖项信息详情

2.3 新建奖项

点击新建奖项，填写奖项信息，上传奖项图片（包括头像），可对图片进行编辑，添加历届奖项信息（关联媒资库中媒资和奖项库中奖项），CP新建奖项后可将奖项保存和提交审核，咪咕人员可将新建奖项后可直接入库

2.4 变更奖项

在奖项详情页修改奖项信息（权限同新建奖项），返回到奖项库主页

2.5 发布奖项

奖项新建或奖项信息变更审核通过后，系统定时将入库的奖项发布，或人工单条或批量发布奖项

2.6 导出奖项记录

将搜索出的奖项记录导出

3.操作指南

3.1 搜索奖项

进入奖项库，点击按钮""可以库中的奖项

3.2 编辑奖项

点击“”编辑当前行奖项

3.3 新增奖项

点击按钮""增加奖项

3.4 发布奖项

点击按钮""预览即将发布的奖项，点击按钮""发布奖项

3.5 导出奖项

点击按钮""导出奖项
