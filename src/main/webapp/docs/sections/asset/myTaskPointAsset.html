点播任务

1.界面截图

进入我的工作台到我的任务下找到点播任务，查看点播媒资申请记录，并可处理点播任务

2.功能说明

2.1 查看点播任务

根据媒资名称、媒资状态、主分类及提交入库时间等条件查询所有组织的点播申请记录，双击进入可查看任务详情

2.2 处理点播任务

双击进入状态为“提交入库”的点播申请任务详情页面，可将该点播申请直接入库或入库驳回，也可在编辑后将该直播申请直播入库或入库驳回

2.3 设置媒资优先级

媒资优先级分为紧急、热点和普通三种，媒资的优先级影响其在转码队列中的次序

2.4 导出点播申请记录

将搜索出的点播申请记录导出

3.操作指南

3.1 搜索申请记录

进入点播媒资申请，点击按钮""可以搜索列表

3.2 搜索点播媒资申请记录

根据媒资名称、媒资状态、主分类、媒资状态及提交入库时间等查询条件搜索此用户所在组织的点播媒资媒资申请记录，具有高级权限的账户可点击选中“”查看所的组织的申请记录。

3.3 查看点播媒资申请记录详情

进入点播媒资申请，双击列表可查看申请详情。

3.4 设置媒资优先级

点击按钮""可以设置媒资优先级

3.5 导出点播媒资申请记录

点击按钮""将搜索出的点播媒资申请记录导出
