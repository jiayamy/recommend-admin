点播媒资管理

1.界面截图

点播媒资管理是对已经审核通过入库的媒资进行管理，可进行查看媒资详情、变更媒资及为媒资绑定版权等操作。

2.功能说明

2.1 查看媒资详情

根据媒资名称、主分类、是否包含媒体文件、内容类别及提交入库时间等条件查询媒资，双击进入可查看媒资详情

2.2 变更媒资

为内容变更版权

2.3 为媒资绑定版权

为媒资绑定一个或多个版权

2.4 导出媒资列表

将搜索出的媒资列表进行导出

3.操作指南

3.1 搜索点播库记录

进入点播采编，点击按钮""可以搜索点播管理列表

3.2 处理详情

双击列表可查看申请详情，点击""可以对媒资信息进行变更，点击""可以为媒资绑定版权，绑定版权的过程同媒资生产绑版权过程。

3.3 查看媒资历史记录

点击按钮""可以查看媒资历史记录。

3.4 导出媒资库记录

点击按钮""将媒资库的记录导出
