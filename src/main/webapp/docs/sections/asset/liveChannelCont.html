直播库

1.界面截图

直播库是对已经入库的内容（媒资+版权）进行管理，包括内容的搜索、查看、发布、撤消发布和导出。

2.功能说明

2.1 查看内容详情

根据内容名称、主分类、频道类型及提交入库时间等条件查询内容，双击进入可查看内容详情，包括媒资详情和版权详情

2.2 发布内容

系统定时或人工将"待发布"的内容批量发布

2.3 撤消发布

将“已发布”状态的内容撤消发布

2.4 导出内容列表

将搜索出的内容列表进行导出

3.操作指南

3.1 搜索直播库记录

进入直播库，点击按钮""可以搜索直播库列表

3.1 查看内容详情

根据内容名称、主分类、是否包含媒体文件、内容类别及提交入库时间等条件查询内容，双击进入可查看内容详情，包括媒资详情和版权详情

3.2 发布内容

点击按钮"人工将"待发布"的内容批量发布

3.3 撤消发布

点击按钮"将“已发布”状态的内容撤消发布

3.4 变更版权

点击按钮"对内容变更版权

3.5 导出内容列表

点击按钮"将搜索出的内容列表进行导出
