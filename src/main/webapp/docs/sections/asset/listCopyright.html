版权暂存区

1.界面截图

版权采编库是对新增保存状态和审核驳回状态版权进行查看、编辑、删除和绑定媒资操作

2.功能说明

2.1 查看版权详情

根据版权名称、授权方式、地域范围及可用状态等条件查询版权，双击进入可查看版权详情

2.2 编辑版权

点击编辑按钮，编辑版权信息后可将版权信息保存、提交入库及入库（仅限咪咕人员）

2.3 查看版权保存记录

查看版权操作记录按钮，查看版权操作历史

2.4 删除暂存库版权

删除暂存库的版权记录

2.5 为版权绑定媒资

点击绑定媒资按钮，查找媒资进行绑定，单片授权方式的版权仅可绑定一个媒资，集权授权和频道授权的可绑定多个媒资。

2.6 导出版权列表

将搜索出的版权列表进行导出
