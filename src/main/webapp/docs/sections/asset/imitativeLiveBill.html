节目编排管理

1.界面截图

节目编排管理是为自制模拟频道编排节目单

2.功能说明

2.1 选择模拟频道

选择已发布的自制模拟频道

2.2 新增节目

根据多种查询每件查找内容库，批量选择内容加入到节目单

2.3 调整节目顺序

拖动调整节目顺序

2.4 某条节目前后插入媒资

在某条节目前面或后面插入节目

2.5 设置是否隐藏节目单

设置某条节目是否要在门户中显示

2.6 删除节目

删除某一条或几条节目

2.7 分发节目单

将编辑好的节目单分发到门户

2.8 复制节目单

选择需复制的节目单，将其复制到目标日期

2.9 导出节目单

将搜索出的某一时间区间的节目单导出

3.操作指南

3.1 选择模拟频道

点击按钮""选择已发布的身制模拟频道

3.2 新增节目

点击按钮"根据多种查询每件查找内容库，批量选择内容加入到节目单

3.3 调整节目顺序

拖动调整节目顺序

3.4 某条节目前后插入媒资

点击按钮"或点击按钮"在某条节目前面或后面插入节目

3.5 设置是否隐藏节目单

设置某条节目是否要在门户中显示

3.6 删除节目

点击按钮"删除某一条或几条节目

3.7 修改节目单

点击按钮"编辑节目单，如果修改时间，可在点击按钮"后，点击点击按钮"重新计算节目时间

3.8 复制节目单

选择要复制历史节目单日期，点击按钮"选择需复制的节目单，点击按钮"复制所需的节目

3.9 导出节目单

点击按钮"将搜索出的某一时间区间的节目单导出
