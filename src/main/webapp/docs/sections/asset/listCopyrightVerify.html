版权任务

1.界面截图

进入我的工作台到我的任务下找到版权任务，查看版权申请记录，并可处理版权任务

2.功能说明

2.1 查看版权任务

根据版权名称、版权状态、版权授权方式、版权地区范围及提交入库时间等条件查询所有组织的版权申请记录，双击进入可查看任务详情

2.2 处理版权任务

双击进入状态为“提交入库”的版权申请任务详情页面，将该版权申请直播入库或入库驳回，也可在编辑后将该版权申请直播入库或入库驳回

2.3 导出版权申请记录

将搜索出的版权申请记录导出

3.操作指南

3.1 搜索申请记录

进入版权申请，点击按钮""可以搜索列表

3.2 搜索版权申请记录

根据查询条件搜索版权任务，具有高级权限的账户可点击选中“”查看所有组织的申请记录。

3.3 查看版权记录详情

进入版权申请，双击列表可查看申请详情。

3.4 查看绑定的媒资

点击按钮""查看此版绑定媒资的详情

3.5 绑定媒资

点击按钮""为版权绑定媒资

3.6 导出版权申请记录

点击按钮""将搜索出的版权申请记录导出
