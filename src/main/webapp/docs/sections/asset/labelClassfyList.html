条目管理

1.界面截图

对条目的增删除改查操作。

2.功能说明

2.1 搜索条目

根据条目名称、主分类、是否包含媒体文件、内容类别及提交入库时间等条件查询媒资，双击进入可查看媒资详情

2.2 添加条目

点击添加按钮，填写条目基本信息和条目标签

2.3 编辑条目

点击编辑按钮，编辑条目基本信息和条目标签

2.4 条目必填项配置

点击条目必填项配置按钮，配置各条目在主分类和内容形态下是否必填

2.5 删除条目

点击删除按钮，删除条目或条目中的枚举值

2.6 导出条目列表

将搜索出的条目列表进行导出

3.操作指南

3.1 搜索条目

进入版权申请，点击按钮""可以搜索条目列表

3.2 新增条目

点击选中“可以新增条目，添加条目信息。

3.3 配置条目必填信息

选中某一条目，点击选中“配置条目在哪个主分类和内容形态下是否必填。

3.4 导出条目记录

点击按钮""将选中的条目记导出
