版权库

1.界面截图

版权库是对审核通过的版权，包括单片授权、集体授权和频道授权进行统计、查看和绑定媒资操作

2.功能说明

2.1 查看版权详情

根据版权名称、授权方式、地域范围及可用状态等条件查询版权，双击进入可查看版权详情

2.2 为版权绑定媒资

点击绑定媒资按钮，为版权绑定媒资库中的媒资，并可查看媒资详情

2.3 查看版权历史记录

查看版权历史操作记录

2.4 导出版权列表

将搜索出的版权列表进行导出

3.操作指南

3.1 搜索版权记录

进入版权库，根据查询条件搜索此用户所在组织的版权记录，具有高级权限的账户可点击选中“”查看所的组织的记录。点击按钮""可以搜索列表

3.2 查看版权记录详情

进入版权，双击列表可查看详情，并可在点击""对版权信息进行变更操作，变更后版权操作过程同版权录入过程。

3.3 查看版权历史记录

点击按钮""查看版权操作历史

3.4 导出版权记录

点击按钮""将搜索出的版权记录导出
