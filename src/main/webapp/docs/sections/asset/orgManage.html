组织管理

1.界面截图

组织管理是对媒资系统中组织进行添加、编辑和删除操作。

2.功能说明

2.1 搜索组织

根据搜索条件搜索所需的组织

2.2 新建组织

点击“添加”，填写父组织属性值，添加子组织，为子组织导入账户

2.3 编辑组织

点击编辑组织属性值

2.4 删除组织和账户

删除父组织、子组织和组织中账户

2.5 导入组织和账户

导入一级组织、二级组织和账户

2.6 导出组织和组织中账户

利用搜索条件，导出所需的一级组织、二级组织和账户

3.操作指南

3.1 搜索申请记录

进入组织管理，输入查询条件，点击按钮""可以搜索组织列表

3.2 新建组织

点击按钮""，填写父组织属性值，添加父组织，点击按钮""添加子组织，为子组织导入账户

3.3 编辑组织

点击按钮""编辑组织属性值

3.4 删除组织和账户

点击按钮""删除父组织、子组织和组织中账户

3.5 导入组织和账户

点击按钮""选择导入类型：一级组织、二级组织和账户，将相应的信息导入。

3.6 导出组织和组织中账户

点击按钮""利用搜索条件，导出所需的一级组织、二级组织和账户
