直播采编

1.界面截图

采编库管理的对象是保存（包括新增时保存和变更时保存状态）和入库驳回状态的直播媒资，可对采编库中媒资进行查看和编辑（填写媒资信息、上传媒资相关图片、绑定版权和媒体文件），编辑后可选择将媒资继续保存、提交入库或入库。

2.功能说明

2.1 查看媒资

根据媒资名称、主分类等条件查询媒资，双击进入可查看媒资详情

2.2 编辑媒资

可对采编库中媒资进行编辑，填写媒资信息、上传媒资相关图片、绑定版权和添加直播源，编辑后可根据权限选择将媒资继续保存、提交入库或入库。

2.3 导出采编库记录

将搜索出的媒资记录导出

3.操作指南

3.1 搜索点播采编记录

进入点播采编，点击按钮""可以搜索列表

3.2 查看记录详情

双击列表可查看申请详情，可对媒资进行编辑、保存操作。

3.4 处理点播采编记录

点击按钮""或""可以提交媒资。

3.5 导出采编库记录

点击按钮""将采编库的记录导出
