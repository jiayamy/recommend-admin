系统参数管理

1.界面截图

系统参数管理是可对系统需配置的参数进行管理

2.功能说明

2.1 添加系统参数

添加系统参数键、值和说明

2.2 编辑系统参数

编辑系统参数键、值和说明

2.3 导出系统参数

导出系统参数键、值和说明

2.4 刷新所有缓存

添加或修改完系统参数后要刷新下缓存

3.操作指南

3.1 添加系统参数

点击按钮""添加系统参数键、值和说明

3.2 编辑系统参数

点击按钮""编辑系统参数键、值和说明

3.3 导出系统参数

点击按钮""导出系统参数键、值和说明

3.4 删除系统参数

点击按钮""删除系统参数键、值和说明

3.5 刷新所有缓存

添加或修改完系统参数后点击按钮"刷新缓存
