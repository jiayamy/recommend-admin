内容申请

1.界面截图

进入我的工作台到组织申请找到内容申请，查看内容申请记录。

2.功能说明

2.1 搜索内容申请记录

进入内容申请，根据内容名称、内容状态及提交入库时间查询此用户所在组织的内容申请记录，查看内容状态、媒资状态和版权状态；必要时也可查看所有组织的申请记录。

2.2 查看内容申请记录详情

双击查看

2.3 内容重新同步

将内容状态为“同步失败”的内容重新同步

2.4 内容下线

将已同步成功的内容下线

2.5 导出点播申请记录

将搜索出的内容申请记录导出

3.操作指南

3.1 搜索申请记录

进入内容申请，点击按钮""可以搜索列表

3.2 搜索内容申请记录

根据媒资名称、媒资状态、主分类、媒资状态及提交入库时间等查询条件搜索此用户所在组织的内容媒资申请记录，具有高级权限的账户可点击选中“”查看所的组织的申请记录。点击选中“”查看所的组织直播申请记录。点击选中“”查看组织的点播申请记录。

3.3 查看内容申请记录详情

进入内容申请，双击列表可查看申请详情，包括媒资信息和版权信息。

3.4 内容重新同步

点击按钮""可将内容状态为“同步失败”的内容重新同步

3.5 内容下线

点击按钮""将已同步成功的内容下线

3.6 导出内容申请记录

点击按钮""将搜索出的内容申请记录导出
