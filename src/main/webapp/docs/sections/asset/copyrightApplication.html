版权申请

1.界面截图

进入我的工作台到组织申请找到版权申请，查看用户及其所在组织的申请记录。

2.功能说明

2.1 查看版权申请记录

根据版权名称、版权状态、版权授权方式、版权地区范围及提交入库时间查询此用户及其所在组织的版权媒资申请记录；具有高级权限的人可以查看到所有组织的申请记录。

2.2 导出版权申请记录

将搜索出的版权申请记录导出

3.操作指南

3.1 搜索申请记录

进入版权申请，点击按钮""可以搜索列表

3.2 搜索版权申请记录

根据媒资名称、媒资状态、主分类、媒资状态及提交入库时间等查询条件搜索此用户所在组织的版权媒资申请记录，具有高级权限的账户可点击选中“”查看所的组织的申请记录。

3.3 查看版权申请记录详情

进入版权申请，双击列表可查看申请详情。

3.4 查看本账户申请记录

点击按钮""可以查看本账户版权申请记录

3.5 查看本组织申请记录

点击按钮""可以查看本组织版权申请记录

3.6 导出版权申请记录

点击按钮""将搜索出的版权申请记录导出
