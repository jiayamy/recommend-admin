版权征集情况查询

1.界面截图

版权征集情况查询是查询征集内容的版权申报记录，以确定是否结束当前征集活动。

2.功能说明

2.1 查看版权申报记录

查看某一征集内容的版权申请数量和各申报版权的详情

2.2 版权评分

版权征集结束后会对所有审核通过的版权的进行评分，有多条申报的征集内容，终止时进行分数比较，最高分为最优版权，若有多个最高分，选定申报时间最靠前的为最优版权，而征集截止前没有申报记录的征集内容，系统延时后第一条申报的版权审核通过为即确定为最优版权

2.3 导出申情况列表

将搜索出的版权申报列表进行导出

3.操作指南

3.1 搜索版权申报记录

进入版权申请，点击按钮""可以搜索列表

3.2 导出版权申报记录

点击按钮""将搜索出的版权申请记录导出
