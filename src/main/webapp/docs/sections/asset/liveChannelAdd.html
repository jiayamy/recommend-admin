直播生产

1.界面截图

进入媒资管理到直播生产，可填写媒资信息（基础属性和扩展属性）、上传媒资相关图片、绑定版权和媒体文件。

2.功能说明

2.1 选择服务类型

选择服务类型：直播和模拟直播。如果选择模拟直播，则需从节目资源管理处选择已发布的模拟直播资源，且频道名称不可再编辑；

2.2 填写媒资基础信息

填写媒资基础信息，包括媒资简称、主分类、关键字、简介、电视台类型、电台类型和自频道类型等，点击保存后才可进行其他操作

2.3 上传媒资图片

可选择从本地和FTP上传媒资图片，并可根据图片尺寸、比例和像素截图

2.4 填写扩展信息

填写根据主分类和媒资内容形态联动的默认的扩展信息后，还可根据需要在条目管理处 配置自定义的扩展信息

2.5 添加媒体文件

选择一定转码策略，可选择添加源文件或转码后文件。若添加的是源文件，媒体文件需符合源文件命名规范，且只能添加一个源文件；若选择的是转码后文件，则需将所选转码下的所有码率文件全部上传

3.操作指南

3.1 媒资查重

CP填写媒资的名称、内容形态、主分类以及国家和地区点击媒资查重按钮，判断此主分类下该名称媒资是否已存在。如果该名称媒资已在采编库，则系统将已存在的信息直接载入，CP可继续编辑该媒资；如果该媒资已在媒资库，CP可以点击按钮"查看管重名媒资。如下图所示，以媒资《蜜蜂少女队》为例：

3.2 填写媒资基础信息

填写媒资基础信息，包括媒资简称、关键字等，点击按钮"才可进行其他操作

3.3 上传媒资图片

点击按钮""可选择从本地和FTP上传媒资图片，并可根据图片尺寸、比例和像素截图

3.4 绑定版权

点击""，出现绑定版权弹框，可以点击 新增版权或直接在输入框查找版权库或版权暂存库的版权， 点击 输入版权名称，查看版权详情后绑定版权

3.5 添加直播源

点击""可以添加直播源，选择
添加直播源地址，添加直播源后，双击修改直播源

3.6 保存媒资信息

点击""可选择保存媒资，媒资将会暂存在采编库，后续可以继续编辑。

3.7 提交媒资

点击""可将媒资提交入库，可去直播申请页面查看已经提交入库的媒资。

3.8 入库媒资

点击""可将媒资直接入库，可去直播申请页面查看媒资状态。
