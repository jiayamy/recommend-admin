人物申请

1.界面截图

进入我的工作台到组织申请找到人物申请，查看用户及其所在组织的申请记录。

2.功能说明

2.1 查看人物申请记录

根据人物各种名称、人物状态、此用户及其所在组织的人物申请记录；具有高级权限的人可以查看到所有组织的申请记录。

2.2 导出人物申请记录

将搜索出的人物申请记录导出

3.操作指南

3.1 搜索人物申请记录

根据人物名称、人物状态、人物状态及提交入库时间等查询条件搜索此用户所在组织的人物人物申请记录，具有高级权限的账户可点击选中“”查看所的组织的申请记录。

3.2 查看人物申请记录详情

进入人物申请，双击列表可查看申请详情。

3.3 导出人物申请记录

点击按钮""将搜索出的人物申请记录导出
