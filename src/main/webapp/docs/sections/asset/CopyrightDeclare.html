版权申报

1.界面截图

版权申报是版权CP根据查询的内容才能申报自己的版权

2.功能说明

2.1 搜索征集列表

根据查询条件搜索征集媒资列表

2.2 上传版权

选中在征集中的媒资上传版权，操作过程同版权录入

2.3 终止申报内容

选中在征集中的媒资点击终止申报，将些媒资的版权征集活动中止。

2.4 导出媒资列表

将搜索出的媒资列表进行导出

3.操作指南

3.1 搜索征集列表

进入版权申请，根据查询条件点击按钮""搜索征集媒资列表

3.2 上传版权

选中在征集中的媒资，点击按钮"上传版权，操作过程同版权录入

3.3 终止申报内容

选中在征集中的媒资点击点击按钮"终止申报，将些媒资的版权征集活动中止。

3.4 导出媒资列表

点击按钮"将搜索出的媒资列表进行导出
