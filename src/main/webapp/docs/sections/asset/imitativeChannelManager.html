频道资源管理

1.界面截图

频道资源管理可以申请或编辑、删除某一模拟直播频道资源。

2.功能说明

2.1 搜索自制频道资源

根据资源名称等条件查询自制频道资源，双击进入可查看资源详情

2.2 新增自制频道资源

填写自制频道基本信息，申请人员可“保存”和“提交入库”，频道管理人员新建后可直接“入库”

2.3 分发自制频道资源

将已经入库的自制频道分发到流服务

2.4 下线自制频道资源

将已经下线的流服务从流服务下线

2.5 导出自制频道资源列表

将搜索出的自制频道资源列表进行导出

3.操作指南

3.1 搜索自频道

根据资源名称等条件，点击按钮"查询自制频道资源，双击进入可查看资源详情

3.2 新增自制频道资源

点击按钮"填写自制频道基本信息，申请人员可“保存”和“提交入库”，频道管理人员新建后可直接“入库”

3.3 提交入库自制频道资源

点击按钮"将新增的或修改后的自制频道提交入库

3.4 入库自制频道资源

点击按钮"将新增或变更的频道资源入库

3.5 删除自制频道资源

点击按钮"将未入库的自频道删除

3.6 导出自制频道资源列表

点击按钮"将搜索出的自制频道资源列表进行导出
