EPG生产

1.界面截图

进入媒资管理到EPG生产页面，为直播频道创建节目单或导入线下创建的节目单，并可发布节目单

2.功能说明

2.1 搜索频道名称

搜索频道名称，为相应的频道生产节目单

2.2 导入节目单

将按指定格式编辑好的节目单文件导入

2.3 编辑节目单

可逐条编辑节目的名称和时间，可保存编辑后的节目单

2.4 发布节目单

将保存后的节目单发布

2.5 搜索节目单

搜索一定时间区间的节目单

2.6 导出节目单

将搜索出的节目单导出

3.操作指南

3.1 搜索频道

进入EPG生产页面，点击按钮""可以搜频道列表

3.2 导入节目单

点击“”选择已经编辑好的节目单，然后点击“”导入节目单。

3.3 编辑节目单

点击“”编辑当前行节目单，点击“”保存当前行节目单，点击“”删除当前行节目单。

3.4 保存节目单

点击按钮""保存当前导入的所有节日单，点击按钮"批量删除节目单

3.5 搜索节目单

点击按钮""搜索节目单

3.6 单条新增节目单

点击按钮""逐条增加节目单

3.7 发布节目单

点击按钮""预览此频道已经发布7天前起的所有节目单，点击按钮""发布节目单

3.8 导出节目单

点击按钮""导出节目单
