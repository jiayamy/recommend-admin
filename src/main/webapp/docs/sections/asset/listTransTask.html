转码任务管理

1.界面截图

转码任务管理是对所有的转码任务进行集中管理，可以查看转码任务的详情，对转码失败的进行重新转码

2.功能说明

2.1 查看转码记录

根据查询条件查看媒体文件的转码状态

2.2 重新执行转码任务

对于转码失败的任务可点击“重新转码”按钮进行重新转码

2.3 查看转码进度详情

选择某一转码中的任务，可查看其各个码率的转码进度

2.4 导出转码任务记录

可导出所需的转码任务记录

3.操作指南

3.1 搜索转码记录

进入转码任务管理，点击按钮""可以搜索列表

3.2 重新执行转码任务

对于转码失败的任务可点击按钮""按钮进行重新转码

3.3 查看转码进度详情

选择某一转码中的任务，点击按钮""可查看其各个码率的转码进度

3.4 导出转码任务记录

点击按钮""可导出所需的转码任务记录
