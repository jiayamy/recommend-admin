版权录入

1.界面截图

进入我的工作台到组织申请找到版权申请，查看用户所在组织的申请记录。

2.功能说明

2.1 绑定版权文件

从本地上传或已上传的版权文件，可多选

2.2 绑定媒资

从媒资库查找媒资进行绑定

2.3 提交版权

集体版权可直接提交，非集体版权必须绑定媒资后才能提交，具有版权审核权限的账户可以直接将新建的版权入库

3.操作指南

3.1 填写版权信息

选择版权授权方式，填写版权信息，点击按钮""为版权绑定媒资库里的媒资，在区上传版权文件，可选择从本地直接上传，也可选择使用已有版权的文件

3.2 保存版权

点击“”将当前版权信息保存到暂存库。

3.3 “提交审核”版权

点击“将版权提交，可以到我的工作台下的版权申请查看。

3.4 “审核通过”版权

点击按钮""有审核版权权限的账户可以直接将其建的版权审核通过
