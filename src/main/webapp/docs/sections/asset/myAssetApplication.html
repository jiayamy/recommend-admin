点播申请

1.界面截图

进入我的工作台到组织申请找到点播申请，查看点播申请记录。

2.功能说明

2.1 查看组织点播申请记录

进入点播申请，根据媒资名称、媒资状态、主分类、媒资状态及提交入库时间查询此用户所在的组织的点播媒资申请记录。

2.2 查看所有组织点播申请记录

根据媒资名称、媒资状态、主分类、媒资状态及提交入库时间查询此用户所在组织的点播媒资申请记录。

2.3 媒体文件重新搬迁

将搬迁失败的媒体文件重新搬迁

2.4 媒体文件重新转码

将转码失败的媒体文件重新转码

2.5 导出点播申请记录

将搜索出的点播申请记录导出

3.操作指南

3.1 搜索申请记录

进入点播申请，点击按钮""可以搜索列表

3.2 查看组织点播申请记录

进入点播申请，双击列表可查看申请详情。

3.3 搜索组织点播申请记录

根据媒资名称、媒资状态、主分类、媒资状态及提交入库时间等查询条件搜索此用户所在组织的点播媒资申请记录，具有高级权限的账户可点击选中“”查看所的组织的申请记录。

3.4 查看转码进度

点击按钮""可以查看转码中记录的转码进度

3.5 移到采编库

点击按钮""将转码失败的记录移到采编库进行修

3.6 导出点播申请记录

点击按钮""将搜索出的点播申请记录导出
