直播申请

1.界面截图

进入我的工作台到直播申请，查看该账户及其所在CP的直播申请。

2.功能说明

2.1 查看直播申请记录

进入直播申请，根据媒资名称、媒资状态、频道主分类、电视台类型、电台类型、自频道类型及提交入库时间查询此用户所在组织的直播媒资申请记录；具有高级权限的人可以看到所有CP的直播申请记录。

2.2 导出直播申请记录

将搜索出的直播申请记录导出

3.操作指南

3.1 搜索申请记录

进入直播申请，点击按钮""可以搜索列表

3.2 搜索直播申请记录

根据媒资名称、媒资状态、主分类、媒资状态及提交入库时间等查询条件搜索此用户所在组织的点播媒资申请记录，具有高级权限的账户可点击选中“”查看所的组织的申请记录。

3.3 查看直播申请记录详情

进入点播申请，双击列表可查看申请详情。

3.4 导出点播申请记录

点击按钮""将搜索出的点播申请记录导出
