奖项申请

1.界面截图

进入我的工作台到组织申请找到奖项申请，查看奖项申请记录。

2.功能说明

2.1 查看奖项申请记录

进入奖项申请，根据奖项名称、奖项状态及提交入库时间查询此用户所在组织的奖项申请记录；必要时也可查看所有组织的申请记录。

2.2 导出点播申请记录

将搜索出的奖项申请记录导出

3.操作指南

3.1 搜索奖项申请记录

根据奖项名称、奖项状态、奖项状态及提交入库时间等查询条件搜索此用户所在组织的奖项奖项申请记录，具有高级权限的账户可点击选中“”查看所的组织的申请记录。

3.2 查看奖项申请记录详情

进入奖项申请，双击列表可查看申请详情。

3.3 查看本账户申请记录

点击按钮""可以查看本账户奖项申请记录

3.4 查看本组织申请记录

点击按钮""可以查看本组织奖项申请记录

3.5 导出奖项申请记录

点击按钮""将搜索出的奖项申请记录导出
