输出源管理

1.界面截图

输出源管理是对各个直播频道不同码率的输出 源地址进行集中管理，可修改输出源，批量导入、导出、删除直播输出源地址。

2.功能说明

2.1 批量导入输出源

导入EXCEL文件，批量导入输出 源

2.2 修改输出源

修改输出源URL和码率

2.3 删除输出 源

删除某一条输出源

2.4 导出输出源

将搜索出的输出源列表进行导出

3.操作指南

3.1 搜索直播源

进入直播源管理，点击按钮""可以搜索列表

3.1 批量导入输出源

点击按钮"选择编辑好的直播源列表，然后点击按钮"导入EXCEL文件，批量导入输出 源

3.2 修改输出源

点击按钮"修改输出源URL和码率，然后点击按钮"保存修改结果

3.3 删除输出源

点击按钮"删除某一条或批量删除输出源

3.4 导出输出源

点击按钮"将搜索出的输出源列表进行导出
