直播收录

1.界面截图

直播收录是建立直播收录任务，对在直播中的节目进行收录，可将收录结果直接发布，也可将收录结果放入素材库

2.功能说明

2.1 新建收录任务

选择直播频道，导入收录任务或手工填写收录任务信息

2.2 删除收录任务

将状态为“未录制”状态的收录任务删除

2.2 重置收录任务

将状态为“收录失败”的收录任务重新收录

2.3 发布收录结果

将状态为“收录完成”且带有媒资ID的收录结果发布

2.4 导出发布任务

选中的收录任务导出

3.操作指南

3.1 搜索申请记录

进入直播收录页面，点击按钮""可以搜索收录任务列表

3.2 新建收录任务

点击按钮""选择直播频道，导入收录任务或手工填写收录任务信息

3.3 删除收录任务

点击按钮""将状态为“未录制”状态的收录任务删除

3.4 修复收录任务

点击按钮""将状态为“收录失败”的收录任务重新收录

3.5 发布收录结果

点击按钮""将状态为“收录完成”且带有媒资ID的收录结果发布

3.6 导出发布任务

点击按钮""选中的收录任务导出
