人物库

1.界面截图

进入媒资管理到资信库再到人物库，对媒资相关人物（明星）进行管理，实现对人物的检索、新建、变更、提交入库、入库和发布操作

2.功能说明

2.1 搜索人物

根据人物名称、人物状态及提交入库时间搜索人物

2.2 查看人物

双击查看人物信息详情

2.3 新建人物

点击新建人物，填写人物信息，上传人物图片（包括头像），可对图片进行编辑，选择关联人物，CP新建人物后可将人物保存和提交审核，咪咕人员可将新建人物后可直接入库

2.4 变更人物

在人物详情页修改人物信息（权限同新建人物），返回到人物库主页

2.5 发布人物

人物新建或人物信息变更审核通过后，系统定时将入库的人物发布，或人工单条或批量发布人物

2.6 导出人物记录

将搜索出的人物记录导出

3.操作指南

3.1 搜索人物

进入人物库，点击按钮""可以库中的人物

3.2 编辑人物

点击“”编辑当前行人物

3.3 新增人物

点击按钮""增加人物

3.4 发布人物

点击按钮""预览即将发布的人物，点击按钮""发布人物

3.5 导出人物

点击按钮""导出人物
