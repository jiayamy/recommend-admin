图片转码管理

1.界面截图

图片转码管理是对各种图片的转码策略进行管理



2.功能说明

2.1 查看图片转码策略

根据查询条件搜索列表中的转码策略

2.2 添加图片转码策略

添加图片转码策略，填写策略名称、策略类别等属性值

2.3 修改图片转码策略

可对已经添加的策略作修改

2.4 导出图片转码策略

可将所需的转码策略导出

3.操作指南

3.1 搜索申请记录

进入版权申请，点击按钮""可以搜索列表

3.2 搜索版权申请记录

根据媒资名称、媒资状态、主分类、媒资状态及提交入库时间等查询条件搜索此用户所在组织的版权媒资申请记录，具有高级权限的账户可点击选中“”查看所的组织的申请记录。

3.3 查看版权申请记录详情

进入版权申请，双击列表可查看申请详情。

3.4 查看本账户申请记录

点击按钮""可以查看本账户版权申请记录

3.5 查看本组织申请记录

点击按钮""可以查看本组织版权申请记录

3.6 导出版权申请记录

点击按钮""将搜索出的版权申请记录导出
