基础导航

点播生产

进入媒资管理到点播生产，可填写媒资信息（基础属性和扩展属性）、上传媒资相关图片、绑定版权和媒体文件。

1. 界面截图

2. 功能说明

2.1 媒资查重

CP填写媒资的名称和主分类，点击媒资查重按钮，判断此主分类下该名称媒资是否已存在。如果该名称媒资已在采编库，则系统将已存在的信息直接载入，CP可继续编辑该媒资；如果该媒资已在媒资库，CP可以直接进行绑定版权操作

2.2 填写媒资基础信息

填写媒资基础信息，包括媒资简称、关键字、简介、字幕语言及国家地区等，点击保存后才可进行其他操作

2.3 上传媒资图片

可选择从本地和FTP上传媒资图片，并可根据图片尺寸、比例和像素截图

2.4 填写扩展信息

填写根据主分类和媒资内容形态联动的默认的扩展信息后，还可根据需要在条目管理处 配置自定义的扩展信息

2.5 添加媒体文件

选择一定转码策略，可选择添加源文件或转码后文件。若添加的是源文件，媒体文件需符合源文件命名规范，且只能添加一个源文件；若选择的是转码后文件，则需将所选转码下的所有码率文件全部上传

2.6 绑定版权

点击绑定版权，输入版权名称，查看版权详情后绑定版权
