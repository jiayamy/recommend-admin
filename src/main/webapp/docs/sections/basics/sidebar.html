布局说明

布局概览

  • 默认的侧边栏布局是:
    1. 折叠按钮
    2. 导航列表（包含侧边栏目录）
    3. 扩展按钮

最小化

  • 折叠按钮可以让侧边栏打开或折叠成最小化状态

屏幕适应

  • 当屏幕尺寸低于992px时，移动视角下有3 种响应式侧边栏。

1.默认移动式菜单

  • 默认移动式菜单

2. 自动最小化式菜单

3. 折叠式菜单

导航折叠按钮

  • 在默认响应式侧边栏时和折叠响应式侧边栏时，折叠按钮用来显示和隐藏侧边栏。
