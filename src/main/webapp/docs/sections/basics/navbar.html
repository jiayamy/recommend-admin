用户菜单

用户菜单

  • 用户菜单有以下功能：
    1. 设置

    点击设置项可以个性化设置首页的显示内容：

    2. 个人资料

    在个人资料处，用户可以链接到用户管管理中心，编辑自己的个人资料

    3. 修改密码

    跳出新页面，进行修改密码操作。

    4. 退出

    退出当前登陆。
