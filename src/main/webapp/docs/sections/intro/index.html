介绍

媒资管理系统是一个集媒资管理、音视频生产、版权管理、内容管理、直播监控为一体的综合管理系统。包含了从媒资（包括点播和直播）的生产、编辑、审核到发布的整个流程以及生产媒资所需的版权、条目、奖项和人物信息的管理。以下对系统中关键术语进行解释：

1.1 媒资内容类型定义

媒体文件：无任何外在描述性信息的纯文件

媒资信息内容：已填写了媒资的基本描述和属性信息（含资信信息），但未绑定任何媒体文件

媒资文件内容：已填写了媒资的基本描述和属性信息，已绑定与当前媒体基本类型相对的媒体文件（如当前基本类型是视频点播，则需绑定视频媒体文件），但未绑定版权信息

版权媒资内容:已填写了媒资的基本描述和属性信息，已绑定与当前媒体基本类型相对的媒体文件，已绑定版权信息

1.2 媒资内容之间的关系

媒资信息=媒资信息内容

媒资信息+媒体文件 =媒资文件内容

媒资文件内容+版权信息 =版权媒资内容

媒资信息+媒体文件+版权信息 =版权媒资内容

1.3 媒资内容对外服务原则

所有的媒资内容（媒资信息内容，媒资文件内容，版权媒资内容）均可对外提供查询和导出， ----------导出指内容可以对外提供，但不可直接用于商业运营

只有版权媒资内容（媒资信息+媒体文件+版权信息）可以对外（商用）输出 ----------输出指内容可以对外提供，也可直接用于商业运营
